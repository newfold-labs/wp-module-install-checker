# Overview

**wp-module-install-checker** checks whether a WordPress installation is a fresh install and fetches the estimated installation date. It registers with the Newfold Module Loader. Used by wp-module-onboarding-data and other modules. No runtime Composer requires. Maintained by Newfold Labs. Distributed via Newfold Satis.
